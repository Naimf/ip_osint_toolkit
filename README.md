# 🔍 IP OSINT & Fraud Investigation Toolkit

A cybersecurity tool for IP geolocation, abuse reputation,
port scanning, and WHOIS lookup — all in one Colab notebook.

## Features
- 📍 Geolocation (city, country, lat/long)
- 🛡️ Abuse reputation scoring (AbuseIPDB)
- 🔎 Open ports & services (Shodan)
- 📋 WHOIS / ASN lookup
- 🗺️ Interactive map output
- 📊 CSV export
